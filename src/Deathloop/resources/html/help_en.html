System Description

This software simulates the ball rolling through loop-the-loop (vertical loop, where a section of track completes a 360 degree circle). The software allows you to adjust the following parameters of the model: starting ramp angle, starting ramp lenght, loop radius and ball radius.

The software allows you to plot graphs of the following functions:

  • Ball speed
  • Ball angular velocity
  • Height (the distance from the lowest point of the ramp to the ball)

User manual

After launching the program, you can see the loading window with the project cover sheet.

The second window is the main application window. You can change the view of the model using the mouse wheel (zoom in, zoom out) and using the left mouse button (press the left mouse button and move the mouse to change the view). Also you can see graphs of various variables. You can change the plot`s scale using the mouse wheel.
